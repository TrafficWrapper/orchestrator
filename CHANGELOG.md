## [0.1.0] - 2026-06-18

### Added

- Initial public orchestrator split with owner admin UI, worker approval,
  per-device bootstrap, Noise_XK enroll/pull, minisign-signed client config,
  APK publication metadata, optional Telegram bot, and isolated signer process.
- Contributor onboarding docs, architecture notes, threat model, security policy
  references, and CI workflow.
- Russian documentation mirrors for existing public docs.

### Changed

- APK publishing docs now point to the app release channel instead of pinning a
  stale APK version/checksum in orchestrator docs.
